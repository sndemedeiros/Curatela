CuratelaAI ⚖️🤖 — O CuratelaAI é uma plataforma criada para auxiliar curadores na elaboração de prestações de contas judiciais, utilizando Inteligência Artificial (Google Gemini) para extrair automaticamente informações de extratos bancários, recibos e notas fiscais e transformá-las em um relatório jurídico organizado e pronto para apresentação. A aplicação possui interface moderna, sincronização automática com Firebase Firestore, geração de PDF, dashboard financeiro e login seguro via Google. O sistema foi desenvolvido com React 19, Vite, Tailwind CSS 4, Google Gemini AI, Firebase, Framer Motion e html2pdf.js. Para usar o projeto localmente, clone o repositório com: git clone https://github.com/seu-usuario/curatela-ai.git e depois entre na pasta com: cd curatela-ai. Instale as dependências com: npm install. Em seguida, renomeie o arquivo .env.example para .env e preencha as chaves do Firebase e a variável GEMINI_API_KEY. Para iniciar o servidor, execute: npm run dev. A aplicação ficará disponível em http://localhost:3000. Para deploy em produção, como na Vercel, configure as variáveis de ambiente VITE_FIREBASE_API_KEY, VITE_FIREBASE_PROJECT_ID, VITE_FIREBASE_APP_ID, VITE_FIREBASE_AUTH_DOMAIN, VITE_FIREBASE_FIRESTORE_DATABASE_ID, VITE_FIREBASE_STORAGE_BUCKET, VITE_FIREBASE_MESSAGING_SENDER_ID e GEMINI_API_KEY diretamente no painel do provedor.
